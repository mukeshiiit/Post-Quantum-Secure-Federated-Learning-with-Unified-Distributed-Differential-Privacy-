# PQ-FL Simulation

Python simulation comparing federated learning algorithms with metrics, plots, and supplementary CSV exports.

## Run

```bash
pip install -r requirements.txt
python main.py
```

Outputs are written to `results/` and `Supplementary_Data_Sheet/`.
